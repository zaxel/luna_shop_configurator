# Next.js + Wix Headless E-commerce with 3D Product Configurator

#### **A project showcasing a headless e-commerce solution with modern 3D configurator features and integrations.**
A headless e-commerce application built with **Next.js** on the frontend and **Wix Headless** on the backend.

This project provides core online-store functionality, including:
- Product listings.
- An administration panel with full control over users, products, orders, and static content
- A user profile with complete account management (orders, addresses, personal information, etc.)
- Payments processed via Stripe, covering the full order lifecycle
- The application is deployed on Vercel (frontend) and Wix (backend).
- 3D Product Configurators
- 3D Sneakers with customizable parts and colors
- 3D T-shirt with support for adding stickers


#### Libraries, tools, and features used:

- [x] Next.js (functional components, client/server components)
- [x] Wix Headless, JWT authentication, custom hooks
- [x] Zustand (state management)
- [x] REST APIs, Axios, localStorage, cookies
- [x] Three.js, @react-three/fiber, @react-three/drei
- [x] Admin panel & user profile (users, products, orders, etc.)
- [x] Tailwind CSS
- [x] And more…


The project is deployed on **Vercel Pages** (frontend) and **Wix** (backend). 

It includes **3D models configurators**:
- 3D Sneakers that you can change parts and set it colors.
- 3D T-shirt with ability to add stickers.
